# Security-checklists
Guias internas y buenas practicas de seguridad para aplicaciones, datos, accesos, gobernanza e infraestructura, basado en OWASP, NIST y lineamientos de seguridad empresarial

#  Security Checklists & Best Practices.
Autor: Javier Toloza  
Propósito: Compilar checklists, buenas prácticas y guías internas para fortalecer la seguridad de aplicaciones, datos y entornos cloud.

------------------------------------------------------

##  Contenido del Repositorio.

### 1. Application Security.
Checklists orientados a:
- Autenticación y autorización
- Validación y sanitización
- Seguridad en sesiones
- Seguridad en headers (CSP, HSTS, X-Frame, etc.)

------------------------------------------------------

### 2. Data Governance.
Buenas prácticas para:
- Clasificación de datos
- Control de accesos y de privilegios mínimos
- Logging y trazabilidad
- Auditorías internas

------------------------------------------------------

### 3. Cloud Security.
Lineamientos para:
- IAM seguro
- Configuración mínima segura (baseline)
- Protección de buckets
- Buenas prácticas de infraestructura

------------------------------------------------------

##  Objetivo
Documentar prácticas esenciales que deben seguir los equipos técnicos para mejorar la postura de seguridad en aplicaciones y datos.
